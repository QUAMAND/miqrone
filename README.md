# Miqrone
다목적용 드론 제작 프로젝트

# 개요
제작자명: `QUAM12`  
제조원: **`Quantum Corporation`**  
- F1: `중앙시 남구, 중앙로 3, 44100`

## Overview
- 분류: `클래스 2` - `전술형`
- 무게: 물 `0.15` 블록 ~ 물 `0.60` 블록(`150kg` ~ `600kg`)
- 최대 고도: ~ `10,000` 블록
- 임무 반경: ~ `2,000` 블록
- 명명 규칙: Mi(다중 임무, `multi mission`) + q(무인)rone(항공기)
   - 코드명: `micro quantum drone`

# 비행
- 이온 엔진: 전기 추진 장치 중 하나로 전하 입자를 고속으로 발사해 추진력을 일으킵니다.
   - 추진 과정
      1. 이온화: 제논(`Xe`)에 전자를 충돌시켜, 양이온으로 전환합니다.
	   2. 전자기 가속: 홀 효과를 이용해 제논 1가 양이온(`Xe⁺`)을 한 방향으로 매우 빠르게 가속합니다.
      3. 이온 분사: 가속된 이온이 엔진 뒤쪽으로 분사되며, 반작용에 의해 위 혹은 앞쪽으로 추진력을 얻습니다.
- 연료: 제논(`Xe`)
   - 생성 종류: 공기 액화법, 천연가스 추출법
      - 공기 액화법: 공기 액화 -> 분별 증류 -> 제논(`Xe`) 정제
         | 장점 | 단점 |
         |------|------|
         | 안정적 | 극히 낮은 농도 |
         | 안전함 |
         | 높은 순도 |
      - 천연가스 추출법: 천연가스 추출 -> 실리카 젤 -> 제논(`Xe`) 정제
         | 장점 | 단점 |
         |------|------|
         | 다소 불안정 | 다소 높은 농도 |
         | 다소 안전하지 않음 |
         | 높은 순도 |
         - 시추기(모래 바닥) -> 천연가스 추출 -> 실리카젤 -> 제논 분리
         - 석탄/목탄 -> 연료 정제기 -> 천연가스 생성 -> 실리카젤 -> 제논 분리
